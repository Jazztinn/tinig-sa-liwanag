# Sugidanon — Recording Script 2

Draft for native-speaker review, then recording. Same format as Script 1
(`docs/recording_script_1.pdf`). When recorded, these map to clip IDs
`hil_cs_041`–`hil_cs_080`.

| Requirement | Details |
|---|---|
| Phone recorder | A phone recorder is okay. Record in a quiet room. |
| Audio format | Save each category as one file; clap loudly after each line so the cut points are clear. We convert to 16 kHz mono wav (`scripts/split_claps.py`). |
| Monolingual controls | Lines 4, 9, 15, 19, 24, 29, 36, 37, 39 are monolingual `[HIL]`. |
| Legend | HIL = Hiligaynon · EN = English · TL = Tagalog |

> Review before recording: confirm every line sounds like natural Ilonggo and
> fix anything that doesn't. These are a draft, not gold.

## 1. Market / Palengke

1. [HIL+EN] Tag-pila ang kilo sang manok subong sa market?
2. [HIL+TL] Tumawad ka anay kay medyo mahal ang prutas.
3. [HIL+TL+EN] Out of stock na ang suka, kaya bumili na lang tayo ng iba.
4. [HIL] Indi malimti dad-on ang sako para sa bugas.
5. [HIL+EN] Pwede mo i-deliver ang order namon buas?

## 2. Transport / Jeepney

7. [HIL+TL] Bilis na tayo kay baka maiwan pa tayo sa byahe.
8. [HIL+TL+EN] Wala na sang jeep, kaya mag-book na lang tayo ng ride.
9. [HIL] Pila ka oras ang biyahe halin diri pa-Iloilo?
10. [HIL+EN] Magbayad ka anay sang exact para indi mabudlayan ang driver.
